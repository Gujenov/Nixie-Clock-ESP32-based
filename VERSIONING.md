MCU.HW_VARIANT.RELEASE_TYPE.BUILD

### Структура:
- **MCU** (1 символ) - семейство контроллера
- **HW_VARIANT** (2 символа) - комплектация железа  
- **RELEASE_TYPE** (1 цифра) - тип релиза
- **BUILD** (2 цифры) - номер сборки

## Детализация компонентов

### 1. MCU Family (контроллер)
| Код | Контроллер | Память | Особенности |
|-----|------------|---------|-------------|
| 1 | ESP32-S3-WROOM-1 | 32MB | Текущая основная платформа |
| 2 | ESP32-S3-MINI | 16MB | Урезанная версия |
| 3 | ESP32-S2 | 16MB | Без Bluetooth LE |
| 4 | ESP32 | 16MB | Базовая версия |

### 2. Hardware Variant (комплектация)
Используется буквенно-цифровая кодировка (A0-Z9)

#### Основные продукты:
| Код | Конфигурация | Индикаторы |      Модули     | Особенности |
|-----|--------------|------------|-----------------|-------------|
| A0 | Полная версия |   6xГРИ    | DS3231, энкодер, кнопка | Все функции |
| A1 |    Без RTC    |   6xГРИ    | Энкодер, кнопка | Без DS3231 |
| A2 | Без будильников | 6xГРИ | DS3231, энкодер | Только время |
| A3 | Базовая | 4xГРИ | Энкодер, кнопка | Основные функции |
| A4 | Минимальная | 4xГРИ | Кнопка | Только время |

#### Специализированные:
| Код | Конфигурация | Индикаторы | Модули | Особенности |
|-----|--------------|------------|---------|-------------|
| B0 | + DCF77 | 6xГРИ | DS3231, DCF77, энкодер | Радиомодуль |
| B1 | Базовая + DCF77 | 4xГРИ | DCF77, энкодер | 4 индикатора + радио |
| C0 | Расширенная | 6xГРИ | DS3231, расширенная панель | Доп. управление |

#### Тестовые/кастомные:
| Код | Назначение |
|-----|------------|
| Z0-Z9 | Тестовые сборки |
| X0-X9 | Кастомные версии |

### 3. Release Type (тип релиза)
| Код | Тип | Назначение | Стабильность |
|-----|-----|------------|--------------|
| 1 | Stable | Продакшен | ✅ Стабильная |
| 2 | Beta | Тестирование | ⚠️ Возможны баги |
| 3 | Alpha | Разработка | 🐛 Нестабильная |
| 4 | Dev | Внутренняя | 🔧 Только для разработки |
| 5 | Custom | Кастомная | 📋 Специальные сборки |

### 4. Build Number (сборка)
- **YYMMDD** - порядковый номер сборки (дата сборки) в рамках релиза
- Автоматически обновляется при каждой сборке

## Примеры версий

### Стабильные релизы:
- `1.A0.1.251201` - ESP32-S3, полная версия 6xГРИ, стабильная, дата 1 дек 25г.
- `1.A3.1.260101` - ESP32-S3, базовая 4xГРИ, стабильная, дата 1 янв 26г.  
- `1.B0.1.260105` - ESP32-S3, 6xГРИ + DCF77, стабильная, дата 5 янв 26г. 

### Тестовые сборки:
- `1.A0.2.251201` - ESP32-S3, полная версия, бета-тест, стабильная, дата 1 дек 25г.
- `1.A3.3.260101` - ESP32-S3, базовая, альфа-версия, стабильная, дата 1 янв 26г.  
- `1.Z0.4.260105` - ESP32-S3, тестовая сборка, стабильная, дата 5 янв 26г. 

## Правила использования

### При смене версии:
1. **Изменение даты** → меняем BUILD
2. **Добавление фичи** → переходим на Beta (2)
3. **Стабилизация** → переходим на Stable (1)  
4. **Смена железа** → меняем HW_VARIANT
5. **Смена контроллера** → меняем MCU
