# Первое подключение, UART и OTA для Nixie Clock

Этот документ описывает порядок первого программирования платы, настройки USB/UART и последующего OTA-обновления прошивки.

## 1. Что нужно для начала

- ESP32-S3 плата, совместимая с `board = esp32-s3-devkitc-1-16mb`.
- USB-C кабель для соединения платы с компьютером.
- Установленный PlatformIO / VS Code.
- Доступ к USB COM-порту на компьютере.
- WiFi с известными SSID/паролями, если требуется OTA или синхронизация времени.

## 2. Первая программная запись через USB

### 2.1. Описание среды PlatformIO

В файле `platformio.ini` настроено две среды:
- `esp32s3_16mb` — обычная USB-загрузка через `esptool`.
- `esp32s3_16mb_ota` — OTA-загрузка через `espota`.

Для первого прошивки используйте `esp32s3_16mb`.

### 2.2. Настройки порта и скорости

В `platformio.ini` указаны:
- `upload_port = COM6` — порт загрузки
- `monitor_port = COM6` — порт для Serial Monitor
- `monitor_speed = 115200` — скорость UART
- `upload_speed = 921600` — скорость прошивки через USB

Если плата подключена на другом COM-порту, замените `COM6` на ваш порт.

### 2.3. Заливка прошивки

1. Подключите плату к компьютеру через USB.
2. Убедитесь, что выбранная среда `esp32s3_16mb` активна.
3. Выполните `Upload` в PlatformIO или команду:
   - `pio run -e esp32s3_16mb -t upload`
4. После успешной загрузки автоматически запускается `scripts/auto_upload_fs.py`, который заливает содержимое папки `data/` в SPIFFS.
5. После завершения в консоли должны появиться сообщения о прошивке и записи файловой системы.

## 3. Открытие UART терминала

### 3.1. Скорость и режим

Откройте Serial Monitor на том же COM-порту, что и загрузка, со скоростью `115200` бод.

### 3.2. Поведение терминала

- Терминал показывает системные сообщения от `Serial.print()`.
- После первого запуска прошивки вы увидите сообщения о загрузке, инициализации времени и режимах.
- В терминале можно вводить команды напрямую.

### 3.3. Команды меню

Из командной строки UART доступны следующие основные команды:
- `em` — войти в инженерное меню.
- `help` или `?` — показать доступные команды текущего меню.
- `back` / `b` — вернуться назад из подменю.
- `out` / `o` — выйти из режима меню.

После ввода `em` откроется инженерное меню и автоматически включится OTA окно на 1 час, если OTA ещё не было запущено.

## 4. Работа с инженерным меню

### 4.1. Вход в инженерное меню

В UART введите:
- `em`

После этого в терминале появится список инженерных пунктов.

### 4.2. Полезные команды

- `help` / `?` — показать текущее меню.
- `back` / `b` — вернуться к предыдущему экрану.
- `out` / `o` — выйти из инженерного меню.
- `ota status` — проверить состояние OTA.
- `ota on` / `ota off` — вручную включить/выключить OTA окно.

## 5. OTA обновление прошивки

### 5.1. Как работает OTA в этой прошивке

- OTA реализована через `ArduinoOTA`.
- Имя хоста: `Clockio-OTA`.
- Порт: `3232`.
- Пароль: `Gujenov-LAB-OTA`.
- OTA окно автоматически открывается на 1 час при входе в инженерное меню.

### 5.2. Настройка environment для OTA

Для OTA используется среда:
- `esp32s3_16mb_ota`

В `platformio.ini` для этой среды указано:
- `upload_protocol = espota`
- `upload_port = Clockio-OTA.local`
- `upload_flags = --timeout=10 --auth=Gujenov-LAB-OTA`

### 5.3. Как запустить OTA-загрузку

1. Закройте Serial Monitor, если он открыт. COM-порт занят и мешает OTA.
2. Убедитесь, что устройство подключено к WiFi и работает OTA-сервер.
3. Выполните команду:
   - `pio run -e esp32s3_16mb_ota -t upload`
4. PlatformIO попытается найти `Clockio-OTA.local` и передать прошивку.

Если mDNS не доступен, в логах UART будет указан IP-адрес устройства. В таком случае можно использовать IP вместо `Clockio-OTA.local`.

### 5.4. Важные замечания

- Если OTA окно не открыто, выполните команду `ota on` через инженерное меню.
- Если устройство не подключено к сети WiFi, OTA не запустится.
- Если перед OTA открыт Serial Monitor, закройте его.
- Если OTA идёт медленно или зависает, смотрите сообщения `Serial` в терминале.

## 6. Как отличить USB-прошивку от OTA

- `esp32s3_16mb` — обычная первая запись через USB.
- `esp32s3_16mb_ota` — удалённое обновление по WiFi.

Для первого запуска всегда используйте `esp32s3_16mb`.

## 7. Быстрый порядок для новичка

1. Подключите плату к USB.
2. Залейте прошивку через `esp32s3_16mb`.
3. Откройте Serial Monitor на `115200`.
4. Введите `em`.
5. Проверьте, что OTA окно открылось и видно статус.
6. Закройте монитор.
7. Выполните OTA через `esp32s3_16mb_ota`.
