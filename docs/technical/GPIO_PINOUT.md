## Используемые пины (47 из 48)

**Итого используется: 26 пинов**

### Группировка по назначению

#### I2C шина (адресное пространство)
- **39 (SDA)**, **40 (SCL)** — DS3231 RTC
- Напряжение: 3.3V
- Скорость: 100 кHz
- Требуется: резисторы подтяжки 10kΩ между SDA/SCL и VCC

#### Энкодер / кнопки (GPIO 1, 2, 7, 8)
- **1 (ENC_A)**, **2 (ENC_B)** — механическое вращение энкодера
- **8 (ENC_BTN)** — кнопка энкодера
- **7 (ALARM_BTN)** — кнопка будильника
- Все с внутренними pullup резисторами (или внешними 10kΩ)
- Активность: LOW (кнопка замыкает на GND)

#### Дисплей Nixie (74HC595 — сдвиговый регистр)
- **5 (SER)**, **6 (SRCLK)**, **4 (RCLK)** — управление сдвигом данных
- **45 (OE)** — выключатель всех выходов регистра
  - LOW = выходы активны (на дисплей)
  - HIGH = выходы отключены (во время загрузки данных в регистр)

#### Управление питанием
- **9 (POWER_FAIL_DETECT)** — сигнал детектора падения питания
- Активность: LOW (питание отказало)

#### microSD (SPI, HSPI)
- **18 (CLK)**, **17 (MOSI)**, **16 (MISO)**, **15 (CS)**
- Стандартный 4-проводной SPI
- CS активен LOW

#### Аудио (I2S, 24-bit)
- **48 (BCLK)**, **47 (LRCLK)**, **38 (DOUT)**
- Выход на цифровой аудиоусилитель (MAX98357A)
- LRCLK: LOW = левый канал, HIGH = правый канал

## Запрещённые пины (НЕ ИСПОЛЬЗОВАТЬ)

### 1. GPIO 35, 36, 37 — Линии памяти (Flash/PSRAM)

```cpp
#define MEM_GPIO_35 35
#define MEM_GPIO_36 36
#define MEM_GPIO_37 37
```

**Почему запрещены:**
- На некоторых модулях ESP32-S3 эти пины задействованы для управления внешней памятью (PSRAM).
- Даже если на вашем конкретном модуле они свободны, использование этих пинов приведёт к нестабильности при обращении к памяти.
- На этапе компиляции будет **ОШИБКА**, если код попытается их использовать.

**Компилятор проверяет:**
```cpp
#if IS_MEM_GPIO(I2C_SDA) || IS_MEM_GPIO(I2C_SCL) || ... 
#error "Forbidden GPIO assignment: GPIO35/36/37 are reserved for memory lines"
#endif
```

---

### 2. GPIO 19, 20 — USB OTG (D-/D+)

```cpp
#define USB_GPIO_DM 19   // USB D- линия
#define USB_GPIO_DP 20   // USB D+ линия
```

**Почему запрещены:**
- На ESP32-S3 с USB-CDC эти пины используются для полнодуплексной коммуникации с хостом.
- Без этих линий USB будет неработоспособен.
- Закрыт обнаружением конфликтов на этапе компиляции:

```cpp
#if IS_USB_RESERVED_GPIO(I2C_SDA) || ...
#error "Forbidden GPIO assignment: GPIO19/GPIO20 are reserved for USB (D-/D+)"
#endif
```

## Свободные пины (возможны для будущего расширения)

| Пин | Замечание |
|-----|----------|
| 0 | Обычно используется для загрузки (BOOT). На некоторых плат используется под внешнюю память. |
| 3 | Иногда используется как TX. Проверьте вашу плату. |
| 10–14 | Потенциально свободны, но требуют проверки специфики вашего модуля. |
| 21–34 | Большинство свободны (проверьте конкретный вариант ESP32-S3). |
| 41–46 | Обычно свободны (кроме уже задействованных). |
| 49–50 | Возможны (но обычно не попадают на стандартный DevKit). |

**⚠️ ВАЖНО:** Перед использованием любого "свободного" пина проверьте документацию вашего конкретного модуля (ESP32-S3 DevKit или кастомной платы). Некоторые пины могут быть зарезервированы производителем.

## Конфликты и проверки на компиляции

Прошивка содержит встроенные проверки пересечений пинов:

### ✓ Проверяемые конфликты:

1. **SR595 (дисплей)**: пины 4, 5, 6, 45 не пересекаются между собой
2. **Энкодер**: пины 1, 2, 8 не пересекаются
3. **Кнопка будильника**: пин 7 не пересекается с энкодером
4. **POWER_FAIL**: пин 9 не пересекается ни с какой другой периферией
5. **Memory GPIO**: пины 35, 36, 37 запрещены для любого использования
6. **USB GPIO**: пины 19, 20 запрещены для любого использования
7. **74HC595 vs microSD SPI**: линии дисплея не пересекаются с SD картой
8. **POWER_FAIL vs Audio I2S**: пин 9 не конфликтует с аудио

### ✗ Если при компиляции видите ошибку:

```
error: "SR595 pin conflict: DATA/CLK/LATCH/OE must be different GPIOs"
error: "Forbidden GPIO assignment: GPIO35/36/37 are reserved for memory lines"
error: "Forbidden GPIO assignment: GPIO19/GPIO20 are reserved for USB (D-/D+)"
```

**Решение:** В `include/hardware.h` найдите конфликтующие `#define` и перепроверьте значения пинов. Возможно, при перепрошивке закралась ошибка.

## Электрические характеристики пинов ESP32-S3

### Стандартные входы (с защитой от ESD)
- **Напряжение:** 0–3.3V
- **Максимальный ток:** ~40 мА на пин
- **Внутренний pullup:** есть на всех GPIO (но используются редко)

### Стандартные выходы (CMOS выход)
- **Напряжение HIGH:** 3.3V (логическая единица)
- **Напряжение LOW:** 0V (логический ноль)
- **Максимальный ток:** ~20 мА на пин (рекомендуется не более 10 мА)
- **Общий лимит по всем GPIO:** ~1.2 А (зависит от сценария)

### Специальные линии
- **I2C (39, 40):** open-drain выходы, требуют external pullup
- **UART (USB):** уровни сигнала через USB мост (автоматический)

## Типичная распиновка разъёмов на плате

### USB-C (питание и UART)
```
Pin 1/2: GND
Pin 4/5: VBUS (5V)
Pin 6: CC
Pin 9/24: GND
```
Serial монитор работает через встроенный USB чип.

### Внешние разъёмы (примерные)

#### I2C RTC (3-pin вилка)
```
1. GND
2. SDA (GPIO39)
3. SCL (GPIO40)
```

#### Энкодер (4-pin разъём)
```
1. GND
2. ENC_A (GPIO1)
3. ENC_B (GPIO2)
4. ENC_BTN (GPIO8)
```

#### Кнопка будильника (2-pin)
```
1. GND
2. ALARM_BTN (GPIO7)
```

#### Дисплей 74HC595 (5-pin SPI-подобный)
```
1. GND
2. SER/DS (GPIO5)
3. SRCLK/SHCP (GPIO6)
4. RCLK/STCP (GPIO4)
5. OE (GPIO45)
```
+ питание 5V/3.3V для самого регистра (отдельно).

#### microSD SPI (5-pin)
```
1. GND
2. CLK (GPIO18)
3. MOSI (GPIO17)
4. MISO (GPIO16)
5. CS (GPIO15)
```

#### Аудио I2S (4-pin)
```
1. GND
2. BCLK (GPIO48)
3. LRCLK (GPIO47)
4. DOUT (GPIO38)
```

## Пример добавления новой периферии

Если нужно добавить, например, IR датчик на GPIO 43:

### В `include/hardware.h` добавьте:
```cpp
#define IR_SENSOR_PIN 43
```

### В функцию проверки конфликтов добавьте:
```cpp
#if (IR_SENSOR_PIN == ENC_A) || (IR_SENSOR_PIN == ENC_B) || ...
#error "Pin conflict: IR_SENSOR_PIN overlaps with existing GPIO"
#endif

#if IS_MEM_GPIO(IR_SENSOR_PIN)
#error "Forbidden GPIO assignment: GPIO35/36/37 are reserved"
#endif

#if IS_USB_RESERVED_GPIO(IR_SENSOR_PIN)
#error "Forbidden GPIO assignment: GPIO19/GPIO20 are reserved for USB"
#endif
```
