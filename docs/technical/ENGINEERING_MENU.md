# Инженерное меню прошивки Nixie Clock

## Назначение
Этот документ описывает инженерное меню, в котором задаётся конфигурация устройства, выполняются аппаратные настройки и отладочное тестирование.

## Реализация
- `src/engineering_menu.cpp`
- `include/engineering_menu.h`
- `src/menu/menu_manager.cpp` — вход в меню и переход к состоянию `MENU_ENGINEERING`
- `src/command_handler.cpp` — команда `em` для входа в инженерное меню

## Вход в инженерное меню
Инженерное меню доступно из основной командной строки через команду:
- `em`

При входе выполняется:
- `inMenuMode = true`
- `printEnabled = false`
- `currentMenuState = MENU_ENGINEERING`
- автоматическое открытие OTA окна на 1 час (`otaEnable(3600UL * 1000UL)`) при условии, что OTA ещё не было включено

## Главный экран инженерного меню
При входе отображается:
- `1  Смена серийного номера`
- `2  Настройки железа`
- `3  Тестирование`
- `4  Сервис и моточасы`

Навигация в меню:
- `back` / `b` — вернуться назад из подменю в инженерное меню
- `out` / `o` — выйти из режима меню
- `help` / `?` — повторно показать текущее меню

## Общая модель
Весь инженерный раздел управляется функцией `handleEngineeringMenu(String command)`.

Если открыт подменю, команда передаётся специализированному обработчику:
- `ENG_SUBMENU_SERIAL` → `handleSerialNumberMenu()`
- `ENG_SUBMENU_HARDWARE` → `handleHardwareMenu()`
- `ENG_SUBMENU_TESTING` → `handleTestingMenu()`
- `ENG_SUBMENU_NIX6_OUTPUT` → `handleNix6OutputMenu()`
- `ENG_SUBMENU_COUNTERS` → `handleCountersMenu()`

## Подменю 1: Смена серийного номера

### Команда
- ввод нового серийного номера длиной ровно 11 символов

### Поведение
- проверка длины
- сохранение в `config.serial_number`
- вызов `saveConfig()`

## Подменю 2: Настройки железа

Доступные команды:
- `1 Nix <кол-во>` / `1 1` / `1 2` / `1 4` / `1 6` — выбрать тип часов Nixie и число разрядов
- `2` / `nix hand` — выбрать тип Nixie hand (2 разряда)
- `3` / `cycl` — выбрать Cyclotron (4 разряда)
- `4` / `vert` — выбрать Vertical
- `5` / `mech pend` — выбрать Mech pend
- `6 1` / `audio 1` — включить аудиоподсистему и будильники
- `6 0` / `audio 0` — отключить аудиоподсистему и выключить будильники
- `7 1` / `button` — выбор управления кнопкой
- `7 2` / `enc` — выбор управления энкодером
- `7 3` / `enc+button` — выбор управления энкодером + кнопкой
- `8 1` / `ir 1` — включить IR датчик движения
- `8 0` / `ir 0` — отключить IR датчик движения
- `9` / `light sens cal` — калибровка датчика освещения (выводит сообщение о разработке)
- `disp fx on` / `10 on` — включить мягкую смену цифр для Nixie часов
- `disp fx off` / `10 off` — отключить мягкую смену цифр

### Особенности Nixie 6
Если выбран режим `1 6` или `nix 6`, открывается дополнительное подменю выбора порядка вывода:
- `1` — стандартный порядок вывода
- `2` — обратный порядок + инверсия битов

При выборе режима Nixie 6 сохраняются значения:
- `config.clock_type = CLOCK_TYPE_NIXIE`
- `config.clock_digits = 6`
- `config.nix6_output_mode` в зависимости от выбора

## Подменю 3: Тестирование

Доступные команды:
- `1` / `sound sd` — воспроизвести звук из microSD
- `2` / `sound flash` — воспроизвести звук из внутренней памяти
- `3` / `sound tone` — воспроизвести тестовый тон 880 Hz
- `4` / `stop` — остановить воспроизведение

Для запуска аудиотестов требуется, чтобы платформа поддерживала звук (`platformGetCapabilities().sound_enabled`).

## Подменю 4: Сервис и моточасы

Доступные команды:
- `1` / `service` — отметить сервисную дату текущей локальной даты
- `2` / `reset` — сбросить все счётчики
- `3` / `exit` — выйти из подменю на главный экран инженерного меню

### Отмеченные параметры
- количество включений: `runtimeCounterGetBootCount()`
- моточасы: `runtimeCounterGetTotalRunSeconds()`
- дата последнего сервиса: `runtimeCounterGetLastServiceDate()`
- моточасы на последнем сервисе: `runtimeCounterGetLastServiceRunSeconds()`

## Важные особенности
- инженерное меню автоматически включает OTA окно на 1 час при входе
- из любого подменю можно выйти назад через `back` / `b`
- при выходе на главный экран инженерного меню подменю сбрасывается в `ENG_SUBMENU_NONE`
