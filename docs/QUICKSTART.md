# Быстрый старт Nixie Clock

Краткий справочник для запуска прошивки и OTA в несколько минут.

## 5-минутный цикл: первая прошивка

### Шаг 1: Подключение
```
Плата ──USB──> Компьютер (COM6 или другой)
```

### Шаг 2: Первая прошивка через USB
```powershell
pio run -e esp32s3_16mb -t upload
```
Ждёте 1-2 минуты. После успеха автоматически заливается файловая система.

### Шаг 3: Открыть Serial Monitor
```
COM6, 115200 бод
```

### Шаг 4: Вход в инженерное меню
```
> em
```

Видите сообщение о включении OTA окна на 1 час.

### Шаг 5: Выход и проверка OTA
```
> ota status
```

Видите статус OTA (`[OTA] ON, окно: NNN сек`).

> **Важно про тайминги OTA.** Вход в инженерное меню (`em`) открывает окно OTA на **1 час**. Ручная команда `ota on` открывает окно только на **5 минут** (значение `OTA_WINDOW_MS` по умолчанию). Проверить остаток окна можно командой `ota status`.

## OTA-обновление

### Предусловия
1. Устройство включено и подключено к WiFi.
2. OTA окно открыто (`ota status` показывает ENABLED).
3. Serial Monitor закрыт.

### Команда
```powershell
pio run -e esp32s3_16mb_ota -t upload
```

## Главные команды меню

| Команда | Что делает |
|---------|-----------|
| `em` | Вход в инженерное меню |
| `?` / `help` | Показать текущее меню |
| `back` / `b` | Вернуться назад |
| `out` / `o` | Выход из режима меню |
| `ota on` | Включить OTA вручную |
| `ota off` | Выключить OTA |
| `ota status` | Статус OTA |

## Структура меню инженера

```
em (вход)
  ├─ 1  Серийный номер
  ├─ 2  Настройки железа (Nixie тип, аудио, IR, etc)
  ├─ 3  Тестирование (звуки, тоны)
  └─ 4  Сервис и моточасы
```

## Если что-то не работает

| Проблема | Решение |
|----------|---------|
| Serial Monitor не видит плату | Проверьте COM-порт в `platformio.ini` |
| OTA не работает | Проверьте WiFi, закройте Serial Monitor, выполните `ota on` |
| Плата не загружается через USB | Нажмите BOOT + RESET, убедитесь в USB кабеле |
